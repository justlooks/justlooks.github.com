用data Pump做数据迁移

Dec 17th, 2012 | Comments

工作中有个从测试数据库到生产数据库的数据迁移需求,因为是新项目,目前生产区和测试区还没有做隔离 所以我考虑利用data pump和database link做数据迁移,只要1条impdp命令就可以实现

首先创建目标库到测试库的database link SQL> create public database link rac1_bm_lk connect to bm_zny identified by “123456” using ‘172.20.101.202:1521/testdb’;

然后赋予目标库帐号imp_full_database角色 (之前帐号需要有connect ,resource角色)

SQL> grant imp_full_database to testme;

进行导入操作 impdp testme/123456 schemas=bm_hch network_link=rac1_bm_lk logfile=bm2rac.log remap_schema=bm_hch:testme remap_tablespace=users:sinopay_u
